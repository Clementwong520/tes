# Flat Rate 分期利息計算器

> 一个轻量、美观、多语言的 Flat Rate 分期利息计算工具，支持自定义货币、汇率换算及结果导出图片。

## 功能特点
- 基于 **Flat Rate** 算法（总利息 = 商品价格 × 总利率）
- 支持 **分期 / 不分期** 切换
- 自定义 **基础货币名称** 和 **目标货币**（汇率换算）
- 两种还款方式：**指定每月还款金额** 或 **指定还款月数**
- 自动计算总利息、分期总额、每月还款额、实际月数等
- 显示详细计算步骤和还款明细表
- 一键 **导出计算结果为 JPG 图片**
- 多语言界面（繁體中文、简体中文、English）

## 如何使用
1. 在浏览器中打开 `index.html` 即可运行，无需安装任何依赖。
2. 填写商品价格、利率、订金等参数，切换分期开关和还款方式。
3. 点击「计算分期」或按 Enter 键查看结果。
4. 如需导出图片，点击「汇出结果图片 (JPG)」。

## 技术依赖
- 纯 HTML / CSS / JavaScript
- 使用 [html2canvas](https://html2canvas.hertzen.com/) 实现图片导出（通过 CDN 加载）

# Flat Rate Installment Interest Calculator

> A lightweight, beautiful, multi-language Flat Rate installment interest calculator with custom currency, exchange rate conversion, and image export.

## Features
- **Flat Rate** algorithm (Total Interest = Price × Rate)
- **Installment / One-time** toggle
- Custom **base currency** and **target currency** (exchange rate support)
- Two repayment modes: **specify monthly payment** or **specify number of months**
- Calculates total interest, total installment, monthly payment, actual months, etc.
- Detailed calculation steps and repayment schedule
- **Export result as JPG image** with one click
- Multi-language UI (Traditional Chinese, Simplified Chinese, English)

## How to Use
1. Open `index.html` in any browser – no installation needed.
2. Fill in price, interest rate, deposit, etc., toggle installment and choose repayment method.
3. Click “Calculate” or press Enter to view results.
4. To export, click “Export Result Image (JPG)”.

## Technical Dependencies
- Pure HTML / CSS / JavaScript
- Uses [html2canvas](https://html2canvas.hertzen.com/) for image export (loaded via CDN)
